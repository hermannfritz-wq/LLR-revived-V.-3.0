# LLR-revived 3.0
### Visual Library Explorer for Logos Bible Software & Verbum

> Inspired by Steve Clark's *Logos Library Reporter* (2015) · [l4libreport.clark-tx.net](https://www.l4libreport.clark-tx.net/)

---

## 🌍 What is this?

**LLR-revived 3.0** is a browser-based library explorer for [Logos Bible Software](https://www.logos.com/) and Verbum. It reads your exported Logos library data and presents all your theological books in a structured, searchable interface — with categories, author filters, related resources, notes, reading plans and direct web research links.

Available in **Deutsch · English · Français · Español** (language switcher top right).

---

## ✨ Features

- 📚 **40+ subject categories** — Systematics, Commentaries, Homiletics, Church History and more
- 🔍 **Full-text search** across all titles and authors
- 👤 **Author filter** — find all works by any author across the entire library
- 🔗 **Related resources** — automatic detection of thematically similar books (by author, Bible book, keywords)
- 📝 **Personal notes** per book — save and export
- 📖 **Reading plans** — track your reading progress
- 🌐 **Web links** per book — Google Scholar, JSTOR, WorldCat, Open Library, Logos Store, Wikipedia
- 🌍 **General theological web resources** — Monergism, Bible Gateway, Blue Letter Bible, Internet Archive and more
- 🖱️ **Right-click context menu** — author books, related resources, web search, open in Logos

## 🚀 Quick Start

### Requirements
- **Python 3.8+** → [python.org/downloads](https://www.python.org/downloads/)
- **Logos Bible Software** (must have been launched at least once)
- **Chrome, Firefox or Edge**

### Installation
1. Download and unzip the package for your language (`LLR-V3.0-DE`, `-EN`, `-FR` or `-ES`)
2. Double-click the installer script:
   - 🇩🇪 `01-INSTALLIEREN-Doppelklick.py`
   - 🇬🇧 `02-INSTALL-DoubleClick-EN.py`
   - 🇫🇷 `03-INSTALLER-DoubleClic-FR.py`
   - 🇪🇸 `04-INSTALAR-DobleClick-ES.py`
3. Open in browser: `LLR-Revived-3_0-XX.html`

### Updating your library
Re-export `data.js` from Logos and replace the old file in the program folder.
